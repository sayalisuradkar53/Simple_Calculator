# Simple_Calculator

A simple and interactive calculator designed to make everyday calculations quick and easy. This project is built with HTML, CSS, and JavaScript, with a clean interface and smooth user interaction.

## ✨ What This Calculator Can Do

- Perform basic mathematical calculations
- Supports addition, subtraction, multiplication, and division
- Delete the last entered number using the backspace option
- Reset the calculator with the clear button
- Use the keyboard to enter calculations
- Works smoothly across different screen sizes

## 💻 Built With

- HTML – Page structure
- CSS – Styling and responsive design
- JavaScript – Calculator logic and functionality

## 📁 Files

- `index.html` – Calculator structure
- `style.css` – Design and layout
- `script.js` – Calculator functionality

## 📌 About the Project

This project was created to practice and demonstrate frontend development skills using HTML, CSS, and JavaScript. It focuses on creating an interactive calculator with a simple and easy-to-use interface.
